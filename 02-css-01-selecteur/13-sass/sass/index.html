Les bases de SASS et SCSS

  • Pour installer SASS, on utilisera NPM.
    npm install -g sass
  • La commande pour compiler du sass/scss en css est
    sass nomDuFichier.scss nomdufichier.css si la commande n'est pas reconnu, tentez de mettre "npx" avant. aller directe sur le terminal du dossier en question
  • Pour éviter de retaper la commande à chaque fois, on peut utiliser l'option "--watch" pour que le terminal reste en surveillance sass --watch nomDuFichier.scss nomdufichier.css
  • Pour surveiller tout un dossier: sass --watch nomDuDossierSource/:nomDuDossierCible/
  • On peut minimiser directement le css avec: sass --watch nomDuDossierSource/:nomDuDossierCible/ --style compressed

SASS

La principale différence entre SASS et SCSS est la syntaxe.
En premier SASS fut créé, avec une syntaxe plus raccourci, se rapprochant de celle de Python, sans accolade ou ";". Elle se base entièrement sur l'indentation.

SCSS

Les fonctionnalités de SASS, plaisant à beaucoup de monde, mais sa syntaxe en rebutant tout autant.
Il fut créé SCSS, qui utilise toute les fonctionnalités de SASS mais avec la syntaxe de CSS.
